# Reversi

An implementation of the game Reversi (Othello) in F#

![reversi](reversi.jpg)

- Install dotnet framework
- Install nodejs

```sh
$ ./reversi.api/dotnet restore
$ ./reversi.api/dotnet run

$ ./reversi.web/npm install
$ ./reversi.web/npm start
```
